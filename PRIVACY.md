# 单词记忆隐私说明

更新时间：2026-07-07

本说明适用于通过 GitHub Release 公开分发的 Android v1.0.0 APK。

## 我们处理哪些数据

“单词记忆”是离线优先的学习工具。当前版本会在用户设备本地保存以下内容：

- 用户手动录入或导入的单词。
- 中文释义、例句、备注和标签。
- 复习卡片、复习时间、复习评分和复习日志。
- App 设置，例如界面语言、默认录入语言、导入重复处理策略。

当前版本不需要注册账号，不要求登录，也没有云同步功能。

## 数据如何保存和使用

上述数据主要保存在设备本地 SQLite 数据库中。应用不会主动把这些学习数据上传到开发者服务器。

当前版本使用本地数据完成以下功能：

- 展示词库和搜索结果。
- 按间隔重复算法安排复习。
- 生成今日新增、今日已练、未来复习压力等统计。
- 导入、导出和恢复用户主动选择的文件。
- 按用户设置安排本地复习提醒。

当前版本未集成广告、第三方分析、账号系统或云端同步。

## 权限说明

应用可能请求以下权限或系统能力：

- 通知权限：用于在单词到期复习时发送本地提醒。关闭通知后，用户仍可手动打开 App 复习。
- 文件选择或文件访问能力：用于用户主动导入 CSV、导入完整备份，或把导出的备份文件分享给其他应用。

## 数据共享和删除

当前版本不会主动向开发者服务器上传学习数据，也不会出售用户数据。

只有在用户主动导出、分享或通过系统文件选择器传递文件时，数据才可能离开本设备。此时用户选择的目标应用或服务可能按照其自身隐私政策处理文件内容。

如果用户需要彻底清除本机数据，可以卸载应用，或在系统设置中清除应用数据。执行前建议先导出完整备份。
