# 单词记忆 Android v1.0.1

这是 Android APK 的一次功能更新，适合已经安装 v1.0.0 的用户覆盖安装。

## 下载文件

- Android APK: `word-memory-1.0.1-android-vc5.apk`
- SHA-256 校验文件: `word-memory-1.0.1-android-vc5.sha256.txt`

## 主要更新

- 新增本地英汉词库建议。
- 输入英文时可一键自动填充中文释义、例句、例句翻译和标签。
- 设置页新增“检查词库更新”，可从公开词库 JSON 下载新增词条并缓存到本机。
- 录入页视觉优化，新增离线词库信息条和建议卡片。

## 安装说明

1. 下载 `word-memory-1.0.1-android-vc5.apk`。
2. 在 Android 设备上打开 APK。
3. 如果系统提示“未知来源应用”，请允许浏览器或文件管理器安装。
4. 安装完成后打开“单词记忆”。

## 校验信息

- SHA-256: `8E8C0D9533EB271D04CA1A5DD56C7A68F57CAEE9A43CA8C8B2B89A5E0AE88550`
- 包名: `com.yujinlon.wordmemory`
- versionName: `1.0.1`

如果设备上已经安装同一签名的 v1.0.0，可以直接覆盖安装；如果安装失败，可能是设备上存在旧签名测试包，需要先卸载旧包。卸载会清除该 App 的本地数据。
