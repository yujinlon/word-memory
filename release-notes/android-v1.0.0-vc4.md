# 单词记忆 Android v1.0.0

这是第一版公开 Android APK，可直接下载并在 Android 设备上侧载安装。当前产物不能直接安装到 macOS。

## 下载文件

- Android APK: `word-memory-1.0.0-android-vc4.apk`
- SHA-256 校验文件: `word-memory-1.0.0-android-vc4.sha256.txt`

## 主要功能

- 英语 / 泰语单词录入
- 间隔重复复习
- 词库搜索和编辑
- CSV 导入导出
- 完整 JSON 备份与恢复
- 本地复习提醒

## 安装说明

1. 在 Android 设备上下载 `word-memory-1.0.0-android-vc4.apk`。
2. 打开 APK。
3. 如果系统提示“未知来源应用”，请允许浏览器或文件管理器安装。
4. 安装完成后打开“单词记忆”。

注意：如果设备上装过旧签名测试包，可能无法覆盖安装，需要先卸载旧包；卸载会清除该 App 的本地数据。
