## 安装

1. 在 Android 设备上下载 `word-memory-1.0.1-android-vc5.apk`。
2. 打开 APK。
3. 如果系统提示“未知来源应用”，请允许浏览器或文件管理器安装。
4. 安装完成后打开“单词记忆”。

如果设备上装过旧签名测试包，可能无法覆盖安装，需要先卸载旧包；卸载会清除该 App 的本地数据。

## macOS

当前 APK 只适用于 Android 设备，不能直接安装到 macOS。

Mac 用户可以下载、校验和转发 APK 到 Android 设备。如果未来发布真正的 macOS 版本，会使用单独的 Release asset 名称。
